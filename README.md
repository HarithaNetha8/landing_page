
# Hari Digital Services Landing Page

## 📌 Project Overview
This project is a simple landing page developed for **Hari Digital Services**, a digital marketing and web services company. The website presents information about the company, its services, team, and contact details in a structured and visually appealing format.

The purpose of this project is to demonstrate the fundamentals of front-end web development and create a professional business landing page that showcases digital services and helps businesses connect with potential clients.

## 🎯 Objectives
- Build a professional landing page for a digital services company
- Present company information and services clearly
- Improve front-end web development skills
- Practice structuring webpages using HTML and CSS

## ✨ Features
- Navigation bar with sections (Home, About, Services, Contact, Team)
- Hero section introducing Hari Digital Services
- About section describing company background and mission
- Services section listing digital marketing services
- Contact section with company contact information
- Clean and organized layout

## 🛠 Services Provided
- Website Development
- Graphic Designing
- Social Media Management
- Digital Marketing
- Content Creation
- Thumbnail Creation
